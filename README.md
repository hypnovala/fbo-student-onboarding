# FBO Student Onboarding

An interactive somatic education app for new students exploring the benefits of somatic bodywork — by **Brock John · Somatic Sex Education**.

## What this is

A calm, guided reflection experience that helps new students:
- Arrive in their bodies before anything else
- Discover which somatic bodywork benefits resonate most
- Build body awareness, sensation vocabulary, and embodied confidence
- Track what shifts over time

This is educational self-awareness work — not therapy, not medical advice.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- react-hook-form + zod
- localStorage persistence (no backend required)
- Vercel-ready

## Getting started

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Checks

```bash
npm run lint
npm run typecheck
npm run build
```

## The seven reflections

1. **Welcome to Your Body** — arrival check-in
2. **Pleasure & Aliveness Map** — locate where you feel alive vs. numb
3. **Sensation Vocabulary Builder** — learn to name what you feel
4. **Desire Discovery** — get honest about what you want
5. **Know Your Nervous System** — read your own regulated vs. activated states
6. **Benefits That Resonate With Me** — identify your most alive entry points
7. **Embodied Confidence Check-In** — track what is shifting over time

## Deployment to Vercel

1. Push the repository to GitHub as `fbo-student-onboarding`
2. Import into Vercel
3. Framework preset: `Next.js`
4. No environment variables required — all data is localStorage-first
